# Figures

Public-safe figure inputs for GitHub, Hugging Face, and slide reuse.

- `architecture.mmd`: Mermaid architecture diagram.
- `RESULTS_SNAPSHOT.md`: Markdown-rendered result table plus Mermaid diagram.
- `results_snapshot.csv`: small machine-readable result table.

These files contain no private database rows, local paths, cluster account
identifiers, or generated model logs.
